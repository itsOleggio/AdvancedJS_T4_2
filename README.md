# Задание 4.3 "Новые типы"
![CI](https://github.com/itsOleggio/AdvancedJS_T4_2/actions/workflows/ci.yml/badge.svg)

## Описание 

Создание колеекции с кодами ошибок и их описанием. И проверкой на то, что код есть 


## Тесты

Используем библиотеку Jest.
Тестом покрыта основная функция программы 
- `add(code, discription) ` - добавление в коллекцию
- `translate(code)` - проверка на то, что код есть в коллекции

Запуск тестов: `npx test`. 

## Код программы

Программа находится в папке `src`.
Тесты ёт в папке `/src/ts/__tests__/`.

## Запуск программы

Сборка программы: `npm build`
Запуск программы: `npm start`.
